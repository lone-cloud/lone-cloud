I build tools I actually use.

---

**[gerbil](https://github.com/lone-cloud/gerbil)**: desktop app for running LLMs on your own hardware.

**[prism](https://github.com/lone-cloud/prism)** + **[prism-android](https://github.com/lone-cloud/prism-android)**: self-hosted notification gateway with Signal, Telegram, and WebPush routing. Android app is a UnifiedPush distributor.

**[polly](https://github.com/lone-cloud/polly)**: minimalist Signal fork. removes stories, donation prompts, and unnecessary UI. arm64-only, Android 14+.

**[snowflake-dashboard](https://github.com/lone-cloud/snowflake-dashboard)**: metrics dashboard for the Tor snowflake proxy.
